Coral Restoration Robot

Fall 2024

Soft Robotics CAD Fabrication

A cephalopod inspired underwater robot powered by a fluidic actuation system as a prototype for low disturbance coral monitoring and restoration efforts.

Overview

Problem

Coral restoration currently requires manual effort from scuba divers, which is difficult, time-consuming, and costly.

Remote monitoring of coral reefs is unreliable, and conventional propellers for locomotion disturb marine creatures.

Approach

Our solution is to develop an underwater robot powered by a fluidic actuation system that can monitor and assist with coral restoration efforts without disturbing marine life.

Results

We've achieved successful tethered underwater motion with a robot built from a fluidic linear actuator, flexible TPU beams, acrylic plates, and a thin flexible outer membrane.

A future goal would be to integrate the control system into the body for untethered motion, possibly using a ring oscillator to reduce the required electronics.

Impact

Our underwater robot would help researchers gather data on coral reefs with less invasive methods by eliminating the need for propellers in a cost-effective and replicatable robotic solution.

Arduino 
const int valve = 8;  // Pin 2
const int motor = 2;  // Pin 3
const int groundPin = 4;  // Pin 4 (ground)
const int groundPin2 = 11;  // Pin 4 (ground)
const int groundPin3 = 9;  // Pin 4 (ground)

const int buttonPin = 12;  // Pin connected to the button


void setup() {
  // Set pin 2 and 3 as outputs
  pinMode(valve, OUTPUT);
  pinMode(motor, OUTPUT);
  
  // Set pin 4 as ground (output set to LOW)
  pinMode(groundPin, OUTPUT);
  digitalWrite(groundPin, LOW);
  pinMode(groundPin2, OUTPUT);
  digitalWrite(groundPin2, LOW);
  pinMode(groundPin3, OUTPUT);
  digitalWrite(groundPin3, LOW);

  pinMode(buttonPin, INPUT_PULLUP);
}

void loop() {
  while (digitalRead(buttonPin) == LOW){
    digitalWrite(motor, HIGH);
    digitalWrite(valve, LOW);
    delay(900);
    
    // Turn pin 3 off
    digitalWrite(motor, LOW);
    digitalWrite(valve, HIGH);
    delay(200); // Wait for 5 seconds
  }
    // Turn pin 3 off
  digitalWrite(motor, LOW);
  digitalWrite(valve, LOW);
  delay(100); // Wait for 5 seconds
}
